# Decisões confirmadas

## Stack proposta
- Frontend: React + TypeScript + Vite
- UI: Tailwind CSS + shadcn/ui
- Swipe: framer-motion
- State: Zustand
- Storage: localStorage
- API: TMDB REST API

## Restrições
- sem backend
- sem autenticação
- sem Supabase
- sem banco externo
- excluir adult=true
- nunca repetir curtidos/descartados na sessão persistida

## Defaults adotados
- catálogo inicial: discover/movie
- idioma default: pt-BR
- região default: BR
- paginação progressiva do TMDB
- fallback para campos ausentes
- tema: dark mode premium/cinematográfico
